# Carpeted Mod

A small Fabric mod that lets you put carpet on vanilla slabs and stairs.

Right-click a supported slab or stair with any carpet to cover it. Sneak-right-click
with an empty hand to remove the carpet and get it back.

Breaking a carpeted block drops both the original block and the carpet.

## Requirements

- Minecraft 26.2
- Fabric Loader 0.19.3 or newer
- Fabric API
- Fabric Language Kotlin
- Java 25 or newer

## Building

```sh
./gradlew build
```

The built jar will be in `build/libs`.
